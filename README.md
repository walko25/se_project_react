## Project Name:

"WTWR" or "What to Wear"

## Functionality:

This app provides users with clothing recomendations based on the current local weather. It breaks down weather patterns into hot, warm, and cold. It pulls the users current location and tracks the weather using a public API.

## Tech Stack:

React, Vite, CSS, HTML, JavaScript
